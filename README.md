# Frenkie the Phoenix

A data analysis of two of Frenkie de Jong's La Liga seasons at FC Barcelona (19/20 & 20/21).

Built with StatsBomb open data, Python, pandas, mplsoccer, and matplotlib.

## Visualizations
- Two-season touch heatmap
- Average position per match
- Pass accuracy and volume over time  
- How deep he played across 68 matches
- Ball recoveries across the same

## Run it
pip install statsbombpy mplsoccer pandas matplotlib
python dejon_full_analysis.py
